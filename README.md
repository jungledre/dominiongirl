#Dominion Girl

### Description
Deck shuffler for the setup phase of the board game Dominion.

### Link to Heroku
<http://dominiongirl.herokuapp.com/>

### Features
* Cheerio.js Web Scraper
* Async.js 
* lodash.js
* Chosen Multiple select
